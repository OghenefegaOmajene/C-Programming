List of Header Files in C

#include (Standard input-output header):
Used to perform input and output operations in C like scanf() and printf().

#include (String header):
Perform string manipulation operations like strlen and strcpy.

#include (Console input-output header) Perform console input and console output operations like clrscr() to clear the screen and getch() to get the character from the keyboard. #include (Standard library header) Perform standard utility functions like dynamic memory allocation, using functions such as malloc() and calloc(). #include (Math header ) Perform mathematical operations like sqrt() and pow(). To obtain the square root and the power of a number respectively. #include(Character type header) Perform character type functions like isaplha() and isdigit(). To find whether the given character is an alphabet or a digit respectively. #include(Time header) Perform functions related to date and time like setdate() and getdate(). To modify the system date and get the CPU time respectively. #include (Assertion header) It is used in program assertion functions like assert(). To get an integer data type in C/C++ as a parameter which prints stderr only if the parameter passed is 0. #include (Localization header) Perform localization functions like setlocale() and localeconv(). To set locale and get locale conventions respectively. #include (Signal header) Perform signal handling functions like signal() and raise(). To install signal handler and to raise the signal in the program respectively #include (Jump header) - Perform jump functions. #include (Standard argument header) Perform standard argument functions like va_start and va_arg(). To indicate start of the variable-length argument list and to fetch the arguments from the variable-length argument list in the program respectively. #include (Error handling header) Used to perform error handling operations like errno(). To indicate errors in the program by initially assigning the value of this function to 0 and then later changing it to indicate errors. - Complex number arithmetic - Floating-point environment - Limits of floating-point types - Format conversion of integer types - Alternative operator spellings - Ranges of integer types - Nonlocal jumps - Signal handling - alignas and alignof convenience macros - UTF-16 and UTF-32 character utilities - Extended multibyte and wide character utilities - Functions to determine the type contained in wide character data
